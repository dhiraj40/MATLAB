# 🌫️ Image Denoising and Dehazing using MATLAB

A MATLAB-based image processing project focused on enhancing image quality by removing noise and haze using classical filtering and restoration techniques.

---

## 📌 Overview

This project demonstrates fundamental concepts of **digital image processing**, specifically:

- Image denoising (removal of random noise)
- Image dehazing (restoration of visibility in hazy/foggy images)

The system applies filtering and enhancement techniques to improve image clarity for better visualization and analysis.

---

## 🚀 Key Features

- 🧹 Noise removal using filtering techniques  
- 🌫️ Haze removal to improve visibility  
- 📷 Image enhancement for better contrast and clarity  
- 🧠 Implementation using core MATLAB functions (no external frameworks)  
- 📊 Visualization of input vs processed images  

---

## 🧠 Algorithms & Techniques Used

### 🔹 Image Denoising
- Gaussian Filtering  
- Spatial smoothing techniques  
- Noise reduction using neighborhood operations  

### 🔹 Image Dehazing
- Contrast enhancement  
- Intensity correction  
- Atmospheric effect reduction  

---

## 🏗️ Workflow

1. Load input image  
2. Apply denoising filters  
3. Apply dehazing techniques  
4. Enhance contrast and visibility  
5. Display original vs processed output
